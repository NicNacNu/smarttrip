Projekt GitHub: https://github.com/NicNacNu/smarttrip

Benötigt: 
	- Python mindestens Version 3.9 
	- Benötigten Bibliotheken, Installiere sie mit: " pip install flask requests python-dotenv reportlab " 

Umgebungsvariablen setzen .env : 
	- AMADEUS_API_KEY= siehe Lanis Nachricht / Email
	- AMADEUS_API_SECRET= siehe Lanis Nachricht / Email
	
Server starten :
	- in der Konsole : " python app.py "

Flask startet dann standardmäßig auf http://127.0.0.1:5000.

Hinweise :
Du nutzt die Test-API von Amadeus (https://test.api.amadeus.com). Die Daten sind nicht live, sondern nur Demo-Angebote.
